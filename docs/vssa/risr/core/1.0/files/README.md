# RISR Core Vocabulary 1.0 / RISR branduolinis žodynas 1.0

## LT

### Paskirtis
Šis paketas apibrėžia **lightweight OWL2** žodyną, skirtą RISR branduoliniam (core) domeno semantikos aprašymui, laikantis **SEMIC Core Vocabulary** principų:
- minimalūs OWL apribojimai (semantika aprašoma, o validacija daroma per SHACL),
- pernaudojami standartai: **SKOS** ir **Dublin Core Terms (DCTERMS)**,
- dvikalbystė: etiketės ir apibrėžtys LT/EN.

### Moduliai
1. **Core ontologija**
   - IRI: `https://data.gov.lt/datasets/gov/vssa/risr/core/1.0/`
   - Failai: `risr-core-1.0.ttl`, `risr-core-1.0.rdf`
2. **API meta ontologija**
   - IRI: `https://data.gov.lt/datasets/gov/vssa/risr/core/1.0/api/`
   - Failai: `risr-api-1.0.ttl`, `risr-api-1.0.rdf`
3. **SHACL validacija**
   - Failai: `risr-core-1.0.shacl.ttl`, `risr-core-1.0.shacl.rdf`
4. **Minimal example data**
   - Failai: `risr-minimal-example.ttl`, `risr-minimal-example.rdf`

## EN

### Purpose
This package defines a **lightweight OWL2** vocabulary for the RISR core domain semantics, aligned with **SEMIC Core Vocabulary** principles.
### Modules
1. **Core ontology**: `risr-core-1.0.ttl`, `risr-core-1.0.rdf`
2. **API meta ontology**: `risr-api-1.0.ttl`, `risr-api-1.0.rdf`
3. **SHACL validation**: `risr-core-1.0.shacl.ttl`, `risr-core-1.0.shacl.rdf`
4. **Minimal example data**: `risr-minimal-example.ttl`, `risr-minimal-example.rdf`
